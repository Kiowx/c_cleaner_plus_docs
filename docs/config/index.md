# 配置说明

## 配置文件位置

所有配置文件位于 `%LOCALAPPDATA%` 目录下：

| 文件 | 用途 |
|------|------|
| `cdisk_cleaner_config.json` | 常规清理勾选状态与拖拽排序 |
| `cdisk_cleaner_custom_rules.json` | 自定义清理规则 |
| `cdisk_cleaner_global_settings.json` | 全局设置（自动保存、更新通道） |
| `%TEMP%\cdisk_cleaner_cache.json` | 硬盘类型检测缓存 |

## 自定义规则格式

### JSON 格式示例

```json
[
  ["规则名称", "目标路径", "目标类型", true, "备注说明"]
]
```

### 字段说明

| 顺序 | 字段 | 类型 | 说明 |
|------|------|------|------|
| 1 | 规则名称 | string | 用于列表显示，建议唯一 |
| 2 | 目标路径 | string | 支持绝对路径和环境变量 |
| 3 | 目标类型 | string | `dir`/`file`/`glob` |
| 4 | 默认勾选 | boolean | true 为默认勾选 |
| 5 | 备注说明 | string | 规则说明文本 |

### 目标类型说明

| 类型 | 说明 |
|------|------|
| `dir` | 清理目录内容 |
| `file` | 清理单个文件 |
| `glob` | 按通配规则清理 |

## 更新通道

| 通道 | 说明 |
|------|------|
| **稳定版** | 经过充分测试的版本，推荐普通用户 |
| **测试版** | 更新较快，适合尝鲜用户 |
```
