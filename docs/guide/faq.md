# 常见问题

## Q1: 程序启动时提示权限不足？

**A:** 右键程序 → 以管理员身份运行。部分清理功能需要管理员权限。

## Q2: 清理后文件还能恢复吗？

**A:**

- 普通模式：文件进入回收站，可恢复
- 强力模式：永久删除，无法恢复

## Q3: 注册表清理安全吗？

**A:** 工具会扫描常见安全路径，但注册表操作存在风险，建议：

- 清理前创建系统还原点
- 备份注册表
- 不确定的项目不要勾选

## Q4: 如何添加自定义清理规则？

**A:** 在常规清理界面直接按说明新建自定义规则  
**B:** 在常规清理界面按格式添加规则  
**C:** 编辑 `%LOCALAPPDATA%\cdisk_cleaner_custom_rules.json`，按格式添加规则。

## Q5: 程序无法检测到某些大文件？

**A:** 检查是否选择了正确的磁盘分区，或调整最小文件大小阈值。

## Q6: 更新后配置丢失？

**A:** 配置文件位于 `%LOCALAPPDATA%`，更新不会覆盖。如丢失可检查该目录。
