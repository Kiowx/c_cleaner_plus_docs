# 项目概述

## 项目简介

**C Cleaner Plus** 是一款专为 Windows 系统设计的 C 盘强力清理工具，采用 Python + Fluent 2 Design 开发，完全开源免费。该工具可扫描并清理 C 盘中的垃圾文件、大文件、重复文件及系统残留，提供友好的 GUI 界面，操作简单直观。

## 项目信息

| 项目 | 信息 |
|------|------|
| 项目名称 | C Cleaner Plus |
| 开发语言 | Python 3.9+ |
| UI 框架 | Fluent 2 Design |
| 支持平台 | Windows 10 / Windows 11 |
| 许可证 | 开源免费 |
| 仓库地址 | https://github.com/Kiowx/c_cleaner_plus |

## 设计理念

- **简洁易用**：直观的 GUI 界面，适合各类用户
- **安全可靠**：完善的警告机制，支持系统还原点
- **功能全面**：覆盖多种清理场景
- **开源透明**：代码完全开放，无隐藏功能

## 系统要求

| 项目 | 要求 |
|------|------|
| **操作系统** | Windows 10 / Windows 11 |
| **Python 版本** | 3.9+（推荐 3.10 / 3.11） |
| **平台支持** | 仅支持 Windows（使用 Windows API） |
| **管理员权限** | 部分功能需要管理员权限 |
| **内存** | 建议 4GB 以上 |
| **磁盘空间** | 建议至少 1GB 可用空间 |

::: tip 提示
Releases 中提供的 `.exe` 文件已包含运行环境，无需额外安装 Python。
:::
