# 使用手册

## 启动流程

```
启动程序
    ↓
检测管理员权限
    ↓
非管理员 → 请求 UAC 提权
    ↓
加载配置文件
    ↓
显示主界面
```

## 常规清理使用

1. 点击「常规清理」选项卡
2. 查看可清理项目列表及预估大小
3. 勾选需要清理的项目
4. 点击「扫描」按钮
5. 确认清理内容后点击「清理」

## 大文件扫描使用

1. 选择要扫描的磁盘分区
2. 设置最小文件大小阈值
3. 点击「开始扫描」
4. 浏览扫描结果，勾选要删除的文件
5. 确认删除

## 高级清理使用

1. 从下拉框选择清理模式
2. 根据提示配置扫描参数
3. 执行扫描
4. 预览结果并确认清理
