# 使用手册

## 启动流程

```
启动程序
    ↓
检测管理员权限
    ↓
非管理员 → 请求 UAC 提权
    ↓
加载配置文件
    ↓
显示主界面
```

## 常规清理使用

1. 点击「常规清理」选项卡
2. 查看可清理项目列表及预估大小
3. 勾选需要清理的项目
4. 点击「扫描」按钮
5. 确认清理内容后点击「清理」

## 大文件扫描使用

1. 选择要扫描的磁盘分区
2. 设置最小文件大小阈值
3. 点击「开始扫描」
4. 浏览扫描结果，勾选要删除的文件
5. 确认删除

## 清理模式

| 模式 | 说明 | 安全性 |
|------|------|--------|
| **普通模式** | 删除文件进入回收站（可恢复） | ⭐⭐⭐ |
| **强力模式** | 永久删除，不进入回收站 | ⭐⭐ |

> ⚠️ 强力模式默认开启，执行前需确认

## 最佳实践

---
1. 首次使用建议先用**普通模式**测试
2. 清理前**创建系统还原点**
3. 不确定的文件不要勾选
4. 定期清理保持系统整洁
