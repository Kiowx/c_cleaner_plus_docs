# 功能特性

## 🔹 常规清理

清理系统日常产生的垃圾文件，支持按需勾选。

| 清理项 | 说明 | 默认勾选 |
|--------|------|----------|
| 用户临时文件 | `%TEMP%` 目录 | ✅ |
| 系统临时文件 | `C:\Windows\Temp` | ✅ |
| Windows 日志 | CBS / DISM 日志 | ✅ |
| 崩溃转储 | Minidump / MEMORY.DMP | ✅ |
| 缩略图缓存 | Explorer 缩略图 | ✅ |
| Shader 缓存 | DirectX / NVIDIA / AMD | ⬜ |
| 浏览器缓存 | Edge / Chrome | ⬜ |
| 包管理器缓存 | npm / Yarn / pnpm / Go / Maven 等 | ⬜ |
| Windows 更新缓存 | 系统更新残留 | ⬜ |

### 支持功能

- 扫描并获取可清理大小
- 按项目勾选执行
- 自定义清理规则

---

## 🔹 大文件扫描

快速定位占用空间较大的文件，帮助释放磁盘空间。

### 扫描配置

- 可选单/多磁盘分区
- 自定义最小文件大小阈值（MB）
- 自定义最大列出数量

### 列表功能

- 按文件大小排序显示
- 显示文件名 / 大小 / 完整路径
- 双击快速勾选
- 单独勾选删除

### 右键菜单

- 复制路径
- 打开所在文件夹
- 在资源管理器中定位

---

## 🔹 高级清理模式

通过下拉框一键切换多种专项清理功能。

### 1. 重复文件查找

采用**三阶段哈希算法**精准定位重复文件：

```
第一阶段 → 文件大小快速筛选
    ↓
第二阶段 → 部分哈希比对
    ↓
第三阶段 → 完整哈希确认
```

**特点：**

- 智能勾选多余副本，保留原始文件
- 支持多分区扫描
- 大幅降低误删风险

### 2. 空文件夹扫描

- 深度遍历指定目录
- 安全清理无实际内容的空目录
- 支持自定义扫描路径
- 扫描结果可预览确认

### 3. 无效快捷方式清理

- 自动解析 `.lnk` 快捷方式
- 找出目标文件已丢失的失效快捷方式
- 支持位置：桌面、开始菜单、快速启动等
- 一键清理无效链接

### 4. 无效注册表扫描

- 清理已卸载软件留下的注册表残留
- 扫描路径：
  - `HKEY_CURRENT_USER\Software`
  - `HKEY_LOCAL_MACHINE\SOFTWARE`
  - 卸载信息残留
- ⚠️ 清理前建议创建系统还原点

### 5. 右键菜单清理

- 深度扫描系统关键注册表位置
- 列出并清理多余、失效的右键扩展项
- 支持递归删除，包含子项的注册表键

## 🔹 应用强力卸载

提供两种卸载路线：

| 模式 | 说明 |
|------|------|
| **标准卸载** | 调用软件自带卸载程序，卸载后可深度扫描残留 |
| **强力卸载** | 面向顽固软件，可解除进程/服务/驱动锁定，强制删除注册表项和文件 |

### 强力卸载特性

- 尝试解除进程、服务与驱动锁定
- 使用系统命令强删注册表项
- 强制删除残留文件与目录
- 内核锁定文件可安排重启后删除

---

## 🔹 清理模式

| 模式 | 说明 | 安全性 |
|------|------|--------|
| **普通模式** | 删除文件进入回收站（可恢复） | ⭐⭐⭐ |
| **强力模式** | 永久删除，不进入回收站 | ⭐⭐ |

> 强力模式默认开启，执行前需确认
