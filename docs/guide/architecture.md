# 技术架构

## 技术栈

```
┌─────────────────────────────────────┐
│           GUI 界面层                 │
│         Fluent 2 Design             │
├─────────────────────────────────────┤
│           业务逻辑层                 │
│    清理引擎 / 扫描引擎 / 卸载引擎     │
├─────────────────────────────────────┤
│           系统接口层                 │
│      Windows API / 注册表操作        │
├─────────────────────────────────────┤
│           数据存储层                 │
│    配置文件 / 缓存文件 / 日志        │
└─────────────────────────────────────┘
```

## 核心模块

| 模块 | 功能 |
|------|------|
| `scanner` | 文件扫描与大小计算 |
| `cleaner` | 文件清理与删除 |
| `registry` | 注册表扫描与清理 |
| `uninstaller` | 应用卸载管理 |
| `hash_checker` | 重复文件哈希比对 |
| `config_manager` | 配置文件管理 |
| `permission` | 权限检测与提权 |
