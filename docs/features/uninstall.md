# 应用强力卸载

提供两种卸载路线：

## 卸载模式对比

| 模式 | 说明 |
|------|------|
| **标准卸载** | 调用软件自带卸载程序，卸载后可深度扫描残留 |
| **强力卸载** | 面向顽固软件，可解除进程/服务/驱动锁定，强制删除注册表项和文件 |

## 强力卸载特性

- 尝试解除进程、服务与驱动锁定
- 使用系统命令强删注册表项
- 强制删除残留文件与目录
- 内核锁定文件可安排重启后删除

## 使用流程

1. 选择要卸载的应用
2. 选择卸载模式（标准/强力）
3. 执行卸载
4. 扫描并清理残留

::: tip 建议
普通软件使用标准卸载即可，顽固软件再使用强力卸载
:::
