# 安全与权限

## 权限管理

- ✅ 启动时自动检测管理员权限
- ✅ 非管理员状态自动请求 UAC 提权
- ✅ 部分功能需管理员权限才能执行

## 安全措施

| 措施 | 说明 |
|------|------|
| 系统还原点 | 可选创建，清理前备份系统状态 |
| 回收站模式 | 普通模式下文件可恢复 |
| 删除警告 | 完善的警告文案，防止误操作 |
| 注册表备份 | 注册表清理前建议手动备份 |

## 安全建议

::: warning 重要提示

1. 清理前**创建系统还原点**
2. 请勿随意删除不明确的大文件
3. 注册表清理前**备份注册表**
4. 首次使用建议先用普通模式测试
5. 重要数据请提前备份

:::

## 清理模式安全性

| 模式 | 说明 | 安全性 |
|------|------|--------|
| **普通模式** | 删除文件进入回收站（可恢复） | ⭐⭐⭐ |
| **强力模式** | 永久删除，不进入回收站 | ⭐⭐ |
